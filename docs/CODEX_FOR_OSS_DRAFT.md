# Codex for Open Source Draft

Use this as starter copy when the public repo is ready.

## Maintainer role

Primary maintainer of `rico-memory-ops`.

## Why this repository qualifies

`rico-memory-ops` is an open toolkit for agent continuity, recall routing, and session handoff. It focuses on a practical gap in long-running AI workflows: not just storing notes, but deciding what memory object should exist, when it should wake up, and what should remain hot vs durable. The repo publishes reusable schemas, templates, and synthetic examples without exposing private user data.

## How API credits would be used

To prototype and evaluate memory ops workflows for open-source agent tooling, including recall routing, continuity handoff templates, synthetic benchmark tasks, and maintainer automation around documentation, issue triage, and release-ready examples.

## Anything else we should know

This project is intentionally derived from real continuity operations, but the published repo contains only the reusable method layer. Personal journals, private task boards, and user-specific memory corpora remain private by design.
