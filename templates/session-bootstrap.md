# Session Bootstrap Template

Use this when a new agent or maintainer needs to pick up the work without rereading everything.

## 1. Current line

- What is the main line right now?

## 2. Already done

- What is already completed and should not be repeated?

## 3. In progress

- What is currently moving?

## 4. Next one thing

- What is the next smallest action that should happen now?

## 5. Private data check

- Does this handoff require real personal context to make sense?
- If yes, do not move it into the public repo.
